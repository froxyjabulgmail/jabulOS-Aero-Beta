JABULOS AERO TECHNICAL PREVIEW

Welcome To jabulOS Aero insider, This build holds all the containing apps (Settings, Start, Task, Terminal And Files)

📦 Getting Started:

JabulOS Needs those specific specs for it to run, Which are
*5MB Of ram
*1 CPU Core, Supports The lowest intel models (and AMD)
*128MB of Additional Storage (no installer yet but if there is in the future release)


🛠️ Technical Specifications

Kernel >	Custom-optimized microkernel architecture
Filesystem	> JFS-A (Jabul File System - Aero)
Memory Management >	Dynamic pre-emptive allocation
Package Manager	> N/A

We like to welcome you to jabulOS Insiders, Use the operating system, Give it a cool rating.

🖥️ How to download the ISO

You can find in the main branch and iso called (jabulOS-Aero.iso), Download it. After that download either VMware Or QEMU, Then create a virtual machine with that ISO


NOTICE: This ISO is LEGACY BIOS Only, Cant run on UEFI Because there isnt any bootloader for this operating system yet!
