Welcome To jabulOS Aero insider, This build holds all the containing apps (Settings, Start, Task, Terminal And Files)

📦 Getting Started:

JabulOS Needs those specific specs for it to run, Which are
*5MB Of ram
*1 CPU Core, Supports The lowest intel models (and AMD)
*128MB of Additional Storage (no installer yet but if there is in the future release)


🛠️ Technical Specifications

Kernel >	Custom-optimized microkernel architecture
Filesystem	> JFS-A (Jabul File System - Aero)
Memory Management >	Dynamic pre-emptive allocation
Package Manager	> N/A
